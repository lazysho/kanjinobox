database setup

konichiwa! welcome to kanji no box! i am glad you find this application useful

before everything else, we need to set up your database. i am very sorry for asking you to do technical stuffs... as of the moment, this is the only way i could think of on how to provide you a storage you can use all by yourself—for no cost

don't worry, if you are interested to know how kanji no box works, i will explain everything later. for now let's start allocating memory space for your box!  

1

Go to Firebase and sign in using your Google Account  

firebase is a lot of things. it has many features that lessens work of developers

kanji no box takes advantage of firebase's authentication feature as well as what you're setting up right now, a database

if you wanna know more about firebase, here

2

Go to your Firebase Console and create a new Firebase Project  

name your project kanjinobox-yourusername changing yourusername to anything you like as long as it isn't your true name. in my case, i wrote sho

by the way your default username is your email, exclusive of @provide.xxx part

3

Create a Database  

there are two kinds of database firebase offers—realtime and firestore.

make sure to choose firestore then for the security rules of your database, select start in test mode

4

Copy Configuration  

this is the crucial part in our setup, its the configuration that will link your project to the database you just created

please, only copy the highlighted part—refer to gif—not the whole configuration. do not include the <script> tags and the last line before </script>

5

Paste Configuration  

paste configuration at the text area at right. at the bottom, add the following lines of code

var userdb = firebase.initializeApp(config, "userdb");

function getUserDatabase() {
return userdb;
}

Configuration

6

Build your Box  

yes, that's it... set up would only take 3 to 5 minutes! from this moment onward, kanji no box will handle everything for you... well except for the security rules or anything that requires opening your firebase account

if you got confused along the way, you can watch the whole set up process below

and when you are ready, just click the build button
